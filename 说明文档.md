# LimiX 通用数据分析平台｜项目说明文档

## 1. 项目规划

### 1.1 项目目标

交付一个可运行的前端原型，演示业务流程与交互闭环，重点展示 AI 驱动的数据分析与决策推理能力。

### 1.2 业务范围

- **项目管理**：多项目并行、团队协作。
- **数据资产**：多源数据导入、预处理、版本管理。
- **决策推理**：时序预测、分类、回归任务配置与结果展示。
- **因果洞察**（新增）：基于细粒度样本切片的深度因果分析。
- **开发空间**：模拟 Jupyter Lab 与终端环境。

### 1.3 核心数据模型 (Mock)

- **Project**: { id, title, status, stats... }
- **Dataset**: { id, title, format, rows, columns, status... }
- **Task**: { id, name, type, status, results... }
- **CausalTask**: { id, name, sourceTaskId, filters, xSpec, ySpec, status, resultRefs... }

## 2. 实施方案

### 2.1 技术栈

- **框架**: React + TypeScript
- **UI**: Ant Design + Vanilla CSS + Tailwind CSS (lucide-react icons)
- **构建工具**: Vite
- **状态管理**: React Context / Hooks
- **Mock**: 本地 `/src/mock/` 模拟数据 + `setTimeout` 模拟异步响应

### 2.2 目录结构

```text
src/
├── components/         # 业务组件
│   ├── causal/        # 因果洞察组件 (NEW)
│   ├── data/          # 数据管理组件
│   ├── tasks/         # 任务管理组件
│   └── ...
├── i18n/               # 国际化配置
├── mock/               # Mock 数据源
├── utils/              # 工具函数
└── App.tsx             # 根路由与布局
```
